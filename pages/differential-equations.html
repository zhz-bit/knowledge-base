微分方程的故事

它不是某人某天为某个问题突然"发明"的,而是 17 世纪研究运动、天体、曲线时,随微积分一起长出来的。核心动机只有一句:已知自然在每一瞬间的变化规律,反推整个系统的过去与未来。

→ 神经网络里的 Neural ODE 线性代数基础 李群 · SLAM
核心思想 之前的困难 牛顿·运动 莱布尼茨·曲线 最初的名题 ODE vs PDE 杨–米尔斯 欧拉·系统化 发展线 理论先行 两种文化 结语

核心思想:求一个"函数",不是一个"数"

普通方程(2x+3=7)的未知量是一个数,解出一个点。微分方程的未知量是一个函数,它给的不是函数本身,而是"函数和它的变化率(导数)之间的关系",解出来是一整条曲线。

为什么需要它?——自然界往往只告诉我们"每一刻怎么变"(速度、加速度、增长率),不直接给结果。微分方程把"变化的规律"写成数学,"解"它就还原出实际的轨迹。最经典的例子:已知一条曲线每点的斜率 dy/dx = 2x,求整条曲线。点画布选一个起点,看它顺着斜率长出来:

点画布任意处 = 选一个过它的解
"解"微分方程 = 积分 = 由各点局部斜率拼回整条曲线。dy/dx=2x 积分得到 y = x² + C —— 同一套斜率规律对应一族曲线(差一个常数 C),初始条件(你点的那个点)挑中其中一条。

换个角度①:解是"一个点" vs 解是"一条曲线"

最直接的区别——点"解一下":左边普通方程收敛成一个数(一个点);右边微分方程长成一个函数(一条曲线)。

换个角度②:求解 = 由"变化率"重建"量本身"(速度 → 路程)

微分方程给你每一刻的变化率(这里是速度 dy/dt = v(t));解它 = 积分 = 把变化率一点点累加,重建出量本身(路程 / 位置)。点播放:阴影面积(∫v)就是车走过的路程。

这就是"求解的意义":现实只给我们"怎么变"(速度、增长率、加速度),微分方程把它写下来;"解"它就是把"变化的规律"积分回"实际的轨迹 / 数值"——从速度重建路程、从加速度重建运动、从斜率重建曲线。

在它之前:静止的代数,挡不住"连续变化"

古代数学擅长"静止"的问题——2x+3=7 求一个固定的数。但 16–17 世纪,科学家开始大量面对"连续变化":

  • 行星为什么沿某条轨道运动?
  • 物体受力后,位置怎样随时间变化?炮弹飞出怎样的轨迹?
  • 一条悬挂的绳索会形成什么曲线?
  • 已知一条曲线每点的切线方向,如何反推出整条曲线?

困难在于:未知的不是一个数,而是一个不断变化的函数。研究行星不能只问"它在哪",还要问"位置如何随时间变化"。而速度是位置的变化率 v = dx/dt,加速度又是速度的变化率 a = d²x/dt²。一旦把物理规律和速度、加速度挂钩,就自然得到含导数的方程 —— 微分方程。

牛顿:从运动与天体出发

牛顿在 1660 年代发展了自己的微积分,称"流数法":随时间变化的量叫"流量"(今天的函数),它的变化速度叫"流数"(今天的导数)。他的目标是描述运动。牛顿第二定律写成现代形式:

m · d²x/dt² = F

物体位置 x(t) 的二阶变化率,由它受的力决定 —— 这就是一个典型的常微分方程。再结合万有引力 F = GMm/r²,就得到关于位置的微分方程;解它,就能解释并预测天体轨道。点"发射"看一颗行星在引力下绕转(拖滑杆改初速:圆 → 椭圆 → 逃逸):

对牛顿,微分方程的核心任务:由作用力 + 初始状态(位置、速度),推算物体未来怎样运动。这套"力 → 加速度 → 数值积分 → 轨迹"的思路,正是今天物理引擎、航天轨道、乃至 Neural ODE 的雏形。

牛顿是怎么"求出"这条轨道的?—— 四步

整个过程是个固定套路,后来所有物理都照着走:受力 → 写出微分方程 → 降阶 → 求解 → 轨迹。

① 受力:万有引力,永远指向地心

地球(质量 M)在原点,卫星位置矢量 r=(x,y)(有方向)、距离 r=|r|=√(x²+y²)(一个数)。引力大小 GMm/r²,方向始终朝向地心:

F = −GMm · r/r³

这两个 r 别混了(你眼尖):r(金色加粗)是矢量 = 位置 (x,y),有方向;r(普通白色)是标量 = 距离 √(x²+y²),就是一个数。于是 r/r³ = (r/r) × (1/r²) = "指向地心的单位方向" × "1/r² 的大小" —— 一个式子同时给了引力的方向与大小。

② F = ma:得到二阶微分方程(卫星质量 m 被约掉)

m·d²r/dt² = −GMm·r/r³　⟹　d²r/dt² = −GM·r/r³

m 消失 = 轻重一起落。拆成两个分量(有心力 → 运动恒在一个平面内,二维足够;下面 r 都是标量距离 √(x²+y²)):

d²x/dt² = −GM·x/r³ 　,　 d²y/dt² = −GM·y/r³

这是二阶方程:加速度由位置决定,只给"规律"。要解它,还需初始条件:t=0 的位置 r₀ 和速度 v₀。

③ 降阶:位置 + 速度 = "状态",变成能一步步算的形式

引入速度 v=dr/dt 当新变量,二阶就拆成一阶方程组,状态 = (x, y, vₓ, v_y):

dx/dt=vₓ , dy/dt=v_y 　|　 dvₓ/dt=−GM·x/r³ , dv_y/dt=−GM·y/r³

④ 一步步算出来:看引力如何"掰弯"速度

每个时间步 Δt 做四件事:① 算 r　② 算加速度 a(指向地心)　③ v += a·Δt　④ x += v·Δt。点"走一步",看 绿 = 速度方向 被 红 = 引力 a(永远指向地心) 一点点拐弯,轨迹就弯成了轨道:

这就是数值积分(演示用稳定的"半隐式欧拉":先更新速度、再用新速度更新位置)。步子 Δt 越小,折线越接近真实光滑椭圆 —— 和"欧拉法 → Neural ODE"完全一个道理。

⑤ 牛顿的解析胜利:轨道一定是"圆锥曲线"

数值能"算出"轨道,但牛顿还证明了它必是圆锥曲线,从而推出开普勒定律。靠两个守恒量:角动量 L=x·v_y−y·vₓ(→ 开普勒第二定律,等面积)和能量 E=½v²−GM/r。用代换 u=1/r,方程化成线性的,解出:

r(θ) = p / (1 + e·cosθ)

这正是圆锥曲线,地球在焦点上(开普勒第一定律)。离心率 e 由初速决定。拖滑杆改 e,看同一个焦点上"圆 → 椭圆 → 抛物线 → 双曲线"的连续变形:

离心率 e轨道初速 v₀
e = 0圆= √(GM/r)(圆速度)
0 < e < 1椭圆(卫星 / 行星,束缚)< 逃逸速度
e = 1抛物线(恰好逃逸)= √2 · 圆速度
e > 1双曲线(逃逸 / 掠过)> 逃逸速度
回到本节顶部那个整轨道演示:滑杆调的初速,就是在改 E、L → 改 e → 圆 / 椭圆 / 逃逸,临界逃逸速度 = √2 × 圆速度 ≈ 1.41 倍(所以它滑到 1.42 就飞走)。

莱布尼茨:从曲线与符号出发

几乎同时,德国数学家莱布尼茨独立发展了微积分。今天我们熟悉的符号 dx、dy、dy/dx、∫ 主要来自他。他关注的是:求曲线切线、由切线反求曲线、求面积、描述两个变化量的关系。

比如一条未知曲线满足 dy/dx = 2x——它没直接告诉你曲线,而是说"每点斜率 = 2x";积分得 y = x² + C(就是上面第一个演示)。微分方程能由各点的局部斜率,恢复整条曲线。莱布尼茨约 1676 年用拉丁文首次使用"微分方程"一词。

视角牛顿莱布尼茨
出发点时间、运动、力学曲线、几何、符号运算
关心物体未来怎么动由斜率反推曲线、变量间关系
留下"流数法"、运动方程dx,dy,dy/dx,∫ 符号、"微分方程"一词

两人之路不同,却共同奠定了微积分与微分方程的基础。

常微分 vs 偏微分:"常"字是后来才加的

这里要澄清一个常见误解:"常微分方程"并不是后来另一个人重新发明的。最早的微分方程,大多就是今天的常微分方程——只有一个自变量,如 dy/dx = f(x,y)。牛顿、莱布尼茨、伯努利兄弟早期研究的,基本都是这一类。当时不必加"常"字,因为另一类还没成熟。

后来人们研究"未知函数同时依赖多个变量"的问题。比如一根琴弦的位移 u(x,t),既依赖位置 x、又依赖时间 t,满足波动方程:

∂²u/∂t² = c² · ∂²u/∂x²

它含偏导数(对一个变量、按住其余),所以叫偏微分方程。18 世纪欧拉、达朗贝尔、拉格朗日研究弦振动、流体、连续介质,PDE 才逐步发展(达朗贝尔用波动方程研究振动弦是代表性工作)。为了区分,才有了"常 / 偏"的叫法。点"拨动",看整根弦(一整片场)随时间振动:

常微分 ODE偏微分 PDE
自变量1 个(如时间 t)2+ 个(空间 x + 时间 t)
导数普通导数 dy/dt偏导 ∂u/∂t、∂u/∂x
解一条曲线一整片场 / 曲面
例子运动、单摆、电路、Neural ODE波动、热传导、流体、扩散模型

所以"常微分方程是谁提出的"没有唯一答案:最早的微分方程本身就是 ODE,"常"只是 PDE 出现后为了区分而加的。

延伸:物理里的"场",就是 PDE 的解

场 = 空间每一点都有值的函数。标量场(温度、电势)每点一个数;矢量场(电场、流速)每点一个箭头。因为场依赖多个变量(x,y,z,t),描述它的物理定律必然含偏导 → 就是 PDE。一句话:粒子的故事用 ODE,场的故事用 PDE。

矢量场:每一点都有一个箭头(点电荷的电场)

拖动电荷(红=正、蓝=负),看每点的电场箭头怎么变 —— 它就是 E ∝ q·r/r³(我们刚讲的"方向 × 平方反比")。满屏的箭头,合起来就是一个矢量场。

杨–米尔斯:物理里"最难的场"

上一节说"场 = PDE 的解"。把这条线推到极致,就是 杨–米尔斯场:它把电磁场推广到支配强力、弱力的层面,是标准模型的骨架;而"严格解出它"至今是百万美元的千禧年难题。下面拆开讲,并尽量可视化。

① 它是什么:电磁场的"会自我作用"升级版(非阿贝尔规范场)

电磁场怎么来的?——规范对称性:带电粒子的量子相位可处处自由改变,为保持物理不变,必须引入一个"补偿场",即电磁场(光子)。这是最简单的规范场论(规范群 U(1),可交换)。杨振宁与 Mills(1954)把它推广到不可交换(非阿贝尔)的群(如 SU(2)、SU(3)):此时"相位"是矩阵、彼此不对易,规范场变成矩阵值的场,并且会和自己相互作用。

② 关键区别:光子互不理睬,胶子彼此纠缠(= 线性 vs 非线性)

麦克斯韦方程是线性的 → 两束光直接叠加、互穿而过;杨–米尔斯方程是非线性的 → 两片场会互相作用。点按钮切换,看两个波包:线性互穿,非线性纠缠:

数学根源是自我作用顶点:QED 只有"电子–光子"顶点,光子之间不直接耦合;QCD 还多了三胶子、四胶子顶点——正是场强里那个 g·[A_μ,A_ν] 对易子项:

③ 它求解什么方程:杨–米尔斯方程(麦克斯韦的非线性推广)

场强(对比电磁 F=∂A−∂A,多了对易子):

F_μν = ∂_μ A_ν − ∂_ν A_μ + g·[A_μ, A_ν]

由作用量 S = −¼∫F·F d⁴x 取极值,得到杨–米尔斯方程(D 是含规范场本身的协变导数):

D_μ F^μν = 0

电磁场(麦克斯韦)杨–米尔斯场
规范群U(1)(可交换)SU(2)、SU(3)(不可交换)
场一个矢量势 A_μ矩阵值的多分量场
方程线性非线性(场乘场,自我作用)
玻色子光子(不带荷、互不作用)胶子 / W、Z(带荷、彼此作用)

④ 为什么"解出它"贡献这么大:从标准模型到夸克禁闭

它是标准模型的骨架:强力 = SU(3) 量子色动力学(胶子粘住夸克)、弱力 = SU(2)(W、Z 玻色子)——四种基本力里的三种都建在杨–米尔斯场上。

自我作用还带来一个惊人后果——夸克禁闭。拖滑杆拉开两个荷,对比上下两块:电磁场线散开、力 ∝ 1/d²(能拉开);强力场线被挤进一条流管、能量 ∝ d(线性增长,永远拉不开,拉太狠流管会"断"成新夸克对):

⑤ 难度阶梯 + 千禧年难题

把"场的难度"排成一道阶梯——杨–米尔斯卡在最高一级:

台阶场 / 方程状态
线性经典电磁场 · 麦克斯韦(线性 PDE)✅ 解得清清楚楚(波、光)
非线性经典杨–米尔斯方程(非线性 PDE)⚠️ 有特殊解"瞬子";Donaldson 用它革新四维拓扑(1986 菲尔兹奖)
非线性 + 量子量子杨–米尔斯(4 维)❌ 千禧年难题:严格"存在性 + 质量间隙"无人能解
千禧年难题(Clay 研究所,百万美元)要两件极深的事:
① 存在性:严格证明四维量子杨–米尔斯理论在数学上真的存在(物理学家天天用、算得极准,但至今没人能严格构造一个 4 维相互作用量子场论);
② 质量间隙:证明最轻激发质量 > 0。经典理论的胶子无质量(像光子),但强力却是短程的、最轻的胶球有质量——质量间隙要从数学上解释这件怪事,它与夸克禁闭紧密相连。
回到主线:它仍是"场 = PDE 的解",只是难度连升两级 —— 线性场(麦克斯韦)→ 非线性经典场(杨–米尔斯,有瞬子)→ 非线性量子场(质量间隙,无人能解)。"解出这个场"= 给整个标准模型一个严格的数学地基、并解释强力为何短程,所以它分量如此之重。

欧拉:把它整理成一门系统的学科

牛顿、莱布尼茨奠基,但真正把常微分方程发展成系统理论的,首推欧拉。从前是"遇到一个具体问题,就想法解一个具体方程";欧拉则去研究规律本身:

  • 方程能分成哪些类型;哪些能变量分离;怎样用积分因子;
  • 高阶、线性方程怎么解;怎样用级数 / 近似求解;
  • 怎样把力学系统普遍写成微分方程。

例如一阶线性方程 dy/dx + P(x)·y = Q(x),可用积分因子 μ = e^∫P dx 化成易积分的形式。他还提出了最基础的数值解法 —— 欧拉法:已知当前状态,沿斜率走一小步推算下一刻

yn+1 = yn + h · f(tn, yn)

这条公式看着眼熟?——它正是ResNet 一层 / Neural ODE 的离散步。想"点着玩"欧拉法的斜率场步进、看 N 步 → Neural ODE 的逼近,见 → 神经网络页的 ◈ 连续节。

一句话分工:牛顿、莱布尼茨开启了微分方程;伯努利家族展示了它解决具体难题的威力;欧拉把常微分方程发展成一门系统学科,并贯通到力学、流体、天体、声学。

一条简化的发展线

时期代表人物主要问题 / 贡献

最准确的结论

  • 微分方程是谁提出的?——产生于牛顿与莱布尼茨创立微积分的过程中:牛顿较早研究"流数方程"(为运动/天体),莱布尼茨建立现代符号并较早用"微分方程"一词(为曲线/变量关系)。不能只归功一人。
  • 常微分方程是谁提出的?——也无唯一提出者。最早那批微分方程本身就是 ODE,欧拉后来系统整理;"常"是 PDE 出现后为区分而加的。
  • 最初为了解决什么?——两大类:物体怎样运动 与 满足某种局部性质的曲线是什么形状;后来才扩展到电路、热学、流体、控制、生物、经济,直到人工智能。
最根本的历史动机:人们已知自然在每一个瞬间遵循什么变化规律,希望由此推算整个系统过去与未来的状态。
三百年后,这个念头换了身衣服又回来了——Neural ODE 用神经网络拟合 dh/dt、扩散模型解概率流、PINN 把 PDE 写进损失:同一个古老的思想。

⭐ 理论先行:从方程里"解"出新世界

物理常被说成"理论领先于实验":因为定律是数学方程,解它、或在对称性 / 一致性下自洽延伸,就会逼出尚未观测的现象——物理学家于是先预言它,再等实验来验证。下面把一批著名预言排成时间线:○ = 理论提出,● = 实验证实,横条 = 苦苦等待的岁月 Δ:

有的立等可取(海王星:算出坐标、当晚就在天上找到);有的等了近一个世纪(引力波 1916→2015 等了 99 年、希格斯 1964→2012 等了 48 年)。共同点:先在纸上(方程里)存在,后在实验里现身。

两条必须的纠偏:① 物理不总是理论先行——黑体辐射、光电效应、暗能量、高温超导都是实验先撞见、理论再追赶;② 理论先行不保证正确——质子衰变、超对称粒子至今没观测到。实验始终是最终裁判。

🪞 两种文化:理论领跑 vs 实验领跑

把上一节的对比推到底:同样是"预言 → 验证"的科学,物理和计算机 / 机器学习常常朝相反方向跑。下图是它们的"镜像"——物理从理论流向实验,CS/ML 从实验流向理论:

物理(尤其基础物理)计算机 / 机器学习
谁领跑理论实验 / 实践
发现流程方程 → 解 → 预言 → 等实验验证试 → 跑 → benchmark 涨就用 → 理论事后解释
为什么有可解的基本定律;实验贵、需理论瞄准;对称 / 一致性逼出新物没有 governing 方程;系统涌现复杂;实验便宜、即跑;只求有用

同一枚硬币的两面。深层原因:物理是发现一个你没造、却有定律的世界(有方程可解 → 理论能抢跑);CS/ML 是造出一个没有先验定律、行为涌现的世界(无方程可解 → 只能造出来测)。但两者都是科学——都要可证伪、都最终由实测裁判。
而我们这一整条线 微分方程 → Neural ODE → 深度学习,恰好横跨了这两种文化:用"解方程"的精神,去研究一个只能"炼丹"出来的对象。

🪜 结语:认识的阶梯——从"记一个特例"到"写一条律"

把整页收成一个念头:物理在找"律"(描述一类的所有情况),而不是"记账"(描述某一个特例)。微分方程就是写"律"的语言——dy/dx=2x 其实就是一个"斜率为 2x 的场":它一口气描述了所有斜率符合这条规律的曲线,而具体的 y=x²+1 只是其中一个成员(C=1)。

沿着 普通方程 → 常微分 → 偏微分 → 统一的场 往上爬,被刻画的对象从"一个数"长成"一条曲线"再长成"一整片场",规律越来越复杂、也越来越难解:

  • 微分方程 = 场:一阶 ODE dy/dx=f(x,y) 就是平面上的方向场,解 = 顺着场流出的积分曲线。它把一族解"求导消去常数"后剩下的共同骨架写了下来(y=x²+C 求导即得 dy/dx=2x)。
  • 统一 = 大贡献:把多个因素并进一个场——电磁 + 弱力 = 电弱(已统一,并预言了 W/Z 玻色子);强力 + 电弱 = 大统一(仍是未竟之梦)。统一用更少的律解释更多世界,还能预言未见。
  • 有了律 ≠ 能解:湍流的场公式(纳维–斯托克斯)我们早就有了,难在解不动(混沌 + 多尺度,其解的存在性本身又是千禧难题)。这正是机器学习切进来"学"它的地方。
  • 地基:这一切的前提是——我们相信现实总满足某种规律,且这规律能用数学写下来(Wigner:"数学不可思议的有效性")。
一句话:普通方程写静态的值,ODE 写一条轴上的演化,PDE 写一整片场的演化;"统一更多因素进一个场"之所以伟大,是因为它用更少的律解释更多世界、还能预言未见——但写下律是一回事,解开它(湍流、杨–米尔斯)又是另一回事。
而我们这一路 微分方程 → Neural ODE → 深度学习,正是用"解方程"的精神,去研究一个只能"炼丹"出来的对象。
