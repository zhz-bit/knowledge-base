① 矩阵 = 一台线性变换机器

一个矩阵,本质上是把整个空间映成新空间的一次线性变换:直线仍是直线、网格仍均匀、原点不动。

本质一:列就是答案

第 i 列 = 第 i 根坐标轴变换后的落点。知道了两列,就知道整个平面怎么动。

本质二:列的重新组合

M·v = v₁·(第1列) + v₂·(第2列)。矩阵乘向量,就是用分量加权叠加它的列。

拖动 / 切换,看 2×2 矩阵怎么搬动整个网格。红 = 第1列(ê₁ 的去向)、绿 = 第2列(ê₂ 的去向)。注意"投影"那档——它把整个平面压成一条线,这正是下一节"降秩"的几何。

② 秩 = 像空间(列空间)的维数

一句话:秩 = 变换之后,空间还剩几维"活着"。

把矩阵作用到整个平面,得到的所有落点构成像空间(= 列张成的空间)。它的维数就是秩:

  • 满秩(2 维):像还是整个平面——上面的旋转 / 缩放 / 剪切,网格仍铺满平面,秩 = 2。
  • 降秩(1 维):像被压成一条线——上面的"投影",整个平面塌到一条直线上,秩 = 1。有一整个方向被碾成了零(这条信息第⑤节会变成"λ=0 的特征向量")。
等价的三种说法(行秩 = 列秩,总相等): 秩 = 列空间维数 = 行空间维数 = 线性无关的列(或行)的个数 = 非零奇异值的个数。

③ 秩的运算法则:乘法压秩 / 加法升秩 / 夹逼

两个关于秩的铁律,以及它们怎么"夹"出一个精确的秩。

乘法压秩:rank(AB) ≤ min(rank A, rank B)

乘法绝不会让秩变大。几何:im(AB) ⊆ im(A)(像只会缩),所以 rank(AB) ≤ rank A;又因 AB 只作用在 im(B) 上,rank(AB) ≤ rank B。像穿不过比它窄的瓶颈。

加法升秩:rank(A+B) ≤ rank A + rank B

加法能让秩变大(最多到两者之和)。几何:(A+B)v = Av + Bv ∈ im(A)+im(B),两个子空间的"和"的维数 ≤ 维数之和。

夹逼:为什么 M = W₁W₂ᵀ 的秩恰好 = d_k

当 A 是 d×d_k、B 是 d×d_k(神经网络那页里的 W_Q、W_K),M = A Bᵀ 是 d×d 却被 d_k 维"细腰"卡住:

d_k (下界·Sylvester 秩不等式) ≤ rank(M) ≤ d_k (上界·乘法压秩 min) ⟹ rank(M) = d_k
拖动 d_k（ambient d = 8）: d_k = 2
串到神经网络那页: 单头注意力 M = W_QW_Kᵀ 是乘法,秩被压死在 d_k;多头则是把 h 个 rank-d_k 的头相加,秩升回 h·d_k = d。乘法压秩 → 低秩瓶颈;加法升秩 → 多头补回。

⑤ 秩 ↔ 特征向量:谁存活,谁被碾平

把第②节的"还剩几维活着"和第④节的特征向量接起来。

核心桥梁是秩-零化度定理加上一个观察:零空间就是 λ=0 的特征空间。

rank(M) = n − dim(零空间)，　而 零空间 = { v : Mv = 0 } = λ=0 的特征空间

所以:

  • 每个 λ=0 的特征向量 = 一个被碾成零的方向(信息没了)。它把秩拉低 1。
  • rank = n −（λ=0 特征空间的维数）= 存活下来的维数。 上面"投影(秩1)"那档:y 轴方向 λ=0 被拍扁到原点,x 轴方向 λ=1 保留 → 1 个活方向 → 秩 = 1。
  • 满秩 ⟺ 0 不是特征值 ⟺ M 可逆 ⟺ det(M) ≠ 0。(det = 所有特征值之积。)
三句精确话(别记错):
· 永远成立:rank = 非零奇异值的个数(σ = √(MᵀM 的特征值))。
· 只对可对角化(尤其对称 / 正规)矩阵:rank = 非零特征值的个数。
· 一般矩阵不成立! 反例:幂零 [[0,1],[0,0]] 秩 = 1,但特征值全是 0(它把平面剪成一条线,却没有"被缩放不偏转"的非零特征方向)。所以"秩 = 非零特征值个数"这句话只在可对角化时安全,通用版要用奇异值。

⑥ 一张体检表:四个指标在说什么

给一个矩阵"体检"的四个量,各回答"这次变换到底做了什么"。

指标直观含义和秩的关系
秩 rank变换后还剩几维活着(像空间维数)—(本表主角)
特征向量只被缩放、不被偏转的方向:Mv=λvλ=0 的特征向量 = 被碾平的方向;它们撑起零空间
特征值 λ对应方向的缩放倍数可对角化时:rank = 非零 λ 的个数;有 λ=0 ⟺ 降秩
奇异值 σ变换在主轴上的拉伸量(σ=√(MᵀM 的特征值))永远:rank = 非零 σ 的个数
行列式 det变换对面积/体积的缩放(带符号)det = ∏λ;det=0 ⟺ 不满秩 ⟺ 0 是特征值

一句话收口: 秩数"几维活着",特征向量找"不被偏转的方向",被碾平的方向(λ=0)正好是丢掉的那几维;通用地数活方向用奇异值,体积是否塌缩看行列式。这套语言,《李群》用它读懂旋转(λ=1 的轴),《神经网络》用它读懂注意力(rank = d_k 的低秩瓶颈)。
