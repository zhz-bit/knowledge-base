→ LeNet(1989/98)
→ AlexNet(2012)
→ VGG / ResNet(14/15)
→ ViT(2020)

序列线(RNN→注意力)

Hopfield / Elman(82/90)
→ LSTM / GRU(97/14)
→ Seq2Seq + Attention(14)
→ Transformer(2017)

生成线

RBM / DBN(2006)
→ GAN / VAE(2014)
→ Diffusion(2020)
→ 文生图 / 大模型(22+)

底座(让上面都成立)

反向传播(1986)
＋ GPU 算力(09+)
＋ 大数据 ImageNet(09)
＋ 规模化 Scaling(20+)

一句话宏观把握: 反向传播给了"可训练"的发动机;CNN 用空间共享拿下视觉、RNN/LSTM 用时间共享拿下序列;2006 DBN 让"深"重新可训、点燃复兴;2012 AlexNet+GPU+大数据引爆;2017 Transformer 用注意力统一序列、再蔓延到视觉与生成,直通今天的大模型。
